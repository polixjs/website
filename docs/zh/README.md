### 开始使用

``` bash
# 安装
npm i polix-cli -g

# 初始化
pol init example

# 编译
make build

# 开发模式启动
make dev

# pm2启动
make start
```

::: warning 注意
请确保你的 Node.js 版本 >= 8。
:::
